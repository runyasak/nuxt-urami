# Installation

1. Install `nuxt-urami` dependency

::: code-group

```sh [pnpm]
$ pnpm add nuxt-urami
```

```sh [npm]
$ npm install nuxt-urami
```

```sh [yarn]
$ yarn add nuxt-urami
```

:::

2. Add `modules` section in `nuxt.config`:

```ts
export default defineNuxtConfig({
  modules: ["nuxt-urami"],
});
```
